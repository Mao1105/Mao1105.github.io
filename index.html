15.1 基本用法

是H5添加的新元素，有几组API构成，具备可了基本绘图能力的2d上下文，还建议了一个名为WebGLde 3D上下文。

基本用法如下：

var canvas=document.getElementById('canvas')
var context=canvas.getContext('2d');//取得绘图上下文
canvas.width=1024;//初始化画布大小
canvas.height=1000;
  • 在使用canvas标签之前需要检测是否有 getContext() 方法
  • if(canvas.getConText()){
    var context=canvas.getContext(‘2d’);//取得绘图上下文
    canvas.width=1024;//初始化画布大小
    canvas.height=1000;
    }

  • 使用toDateURL() 方法可以导出绘制的图像，例如：

    //取得图像URL
var imgURL=canvas.toDateURL('images/png')；
//显示图像
var img=document.createElement('img');
img.src='imgURL';
document.body.appendChild(img);

15.2 2D上下文

15.2.1 基本操作分别是：==填充==和==描边==

  • 填充：颜色/渐变/图像

  • 描边：画线

    fillStyle / strokeStyle=字符串/渐变对象/模式对象

    context.strokeStyle="#000";//绘制的笔画的颜色
context.fillStyle="#ccc";//填充颜色

15.2.2 绘制矩形

  • fillRect():只填充颜色

  • strokeRect():只描边的颜色

  • clearRect():清除一块矩形

    均传入4个参数：
    矩形X坐标，矩形Y坐标，矩形长度，矩形宽度
//先绘制了粉色矩形
    context.fillStyle='pink';
    context.fillRect(10,10,100,50);//只能指定填充色
    //再绘制了透明的紫色矩形
    context.fillStyle='rgba(0,0,255,0.5)';
    context.fillRect(100,20,50,100)

    //绘制蓝色比边框矩形
    context.strokeStyle='#33ccff'
    context.lineWidth=10;
    context.lineCap='square';//线条末端形状：butt-平头 round-圆头 square-方头
    context.lineJoin='round'//线条相交方式：圆交-round 斜交-bevel 斜接-miter
    context.strokeRect(100,150,100,50)//只能指定边框颜色
    //再绘制粉色透明边框矩形
    context.strokeStyle='rgba(255,0,255,0.5)'
    context.strokeRect(100,260,100,50) 

    //清除一块矩形
    context.clearRect(10,10,20,20)

15.2.3绘制路径

  • 首先开始绘制一段路径要调用beginPath() 方法；

  • 随后调用方法创建路径

    1. arc(圆心X坐标，圆心Y坐标，半径，起始点(0)，终点(pi),是否逆时针，默认否)
    2. arcTo(x1,y1,x2,y2,radius):从x1,y1到x2,y2，以radiud为半径以弧形连接两点
    3. moveTo(x,y):将绘图光标移动到x,y。不画线。
    4. lineTo(x,y)：从上一点画直线到x,y点。
      5.rect(x,y,width,height):从x,y画一个矩形，他不是独立图像，只是举行路径。
//绘制一个静态时钟
    clock();
    function clock(){
    //开始路径
        context.beginPath();
    //绘制外圆
        context.arc(100,100,100,0,2*Math.PI);
        context.moveTo(194,100);
    //绘制内圆
        context.arc(100,100,95,0,2*Math.PI);
    //绘制分针
        context.moveTo(100,100)
        context.lineTo(100,15)
    //绘制时针
        context.moveTo(100,100)
        context.lineTo(35,100)
    //描边
        context.stroke();
        //判断某点是否在路径上
        if(context.isPointInPath(100,100)){
            alert('point(100,100)在路径上！')
        }
    }
  • [x] 判断某点是否在路径上用方法：*context.isPointInPath(100,100)

15.2.4绘制文本

  • 方法：fillText()和strokeText();

  • 属性：

    font:’10px solid #ccc Arial’;
    tectAlign:’center’;
    textBaseline:’middle’

      //绘制文本
  text();
  function text(){
  context.font='100px solid Arial';
  context.textAlign='center';//居右-start 居左-end
  context.textBaseline='middle';//居上-bottom 居下-top
  context.fillText('12',100,300)
}

  • 寻找合适文字大小：measureText(“Hello world!”).width

     //在固定区域内，寻找合适的字体大小
findSize();
function findSize(){
    var fontSize=100;//预先设定一个字体的大小
    context.font=fontSize+'px Arial';
    //此时固定区域的长度为140px ,则一直缩小文字，直到文字长度<=140,此时返回文字的height,即文字大小
    while(context.measureText("Hello world!").width>140){
        fontSize--;
        context.font=fontSize+'px Arial';
        console.log(fontSize)
    }
    context.fillText('Hello world!',800,10);
    context.fillText('fonte size is'+fontSize+'px',800,50)
}
